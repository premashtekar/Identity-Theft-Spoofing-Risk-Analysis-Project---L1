🛡️ Identity Theft & Spoofing Risk Analysis
📌 Project Overview
A cybersecurity analytics system that detects suspicious login activities and identity spoofing attempts using rule-based risk scoring.

🎯 Key Features
Impossible Travel Detection - Flags logins from distant locations within short timeframes

Device Spoofing Detection - Identifies rapid device switching

Odd Hour Login Detection - Catches unusual login patterns

Risk Scoring - Assigns HIGH/MEDIUM/LOW risk levels per user

Interactive Dashboard - Visualizes attack patterns and trends

🛠️ Tech Stack
Python (Pandas, NumPy, Matplotlib)

MySQL (Optional)

Streamlit/Power BI (Dashboard)

📊 Outputs
Login dataset with risk flags

Risk alerts with detailed analysis

User risk summary report

Visual dashboards
